# Gadget Shop - Full Stack E-commerce

## Multi Role e-commerce web Application

A **Multi-Role E-commerce Web Application** built with the **Full stack** (MongoDB, Express.js, Next.js, Node.js).  
The system includes **product buy/sell**, **reseller management**, **inventory control**, **referral system**, and a **light/dark mode toggle** for enhanced user experience.  
An integrated **Admin Dashboard** allows full control over users, products, and orders.
